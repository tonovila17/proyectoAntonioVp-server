Pantalla de Configuración da conexión.


Nesta pantalla podemos almacenar nun arquivo externo os axustes cos que o programa tratará de conectarse á base de datos cando pulsemos o botón Conectar na pantalla de Inicio.

Os datos para cubrir, cada un no seu respectivo campo serán: Ip, Porto, Nome da base de datos, Usuario e Contrasinal..

Unha vez enchidos os campos, poderemos gardalos no arquivo externo que almacena a nosa configuración a través do botón Gardar situado na parte inferior, se o proceso complétase correctamente, seremos informados por pantalla mediante unha mensaxe e redirixidos á pantalla de Inicio para poder probar a nosa configuración. Se houbese algún erro no formulario á hora de cubrir os datos, mostraríansenos os motivos polos cales non se puideron gardar os cambios.

Probar Valores por defecto: Se descoñecemos como debemos configurar a conexión, dispoñemos do botón "Probar Conexión con Valores por Defecto", que tratará de establecer a conexión cuns valores fixados mediante código no programa, estes valores son: "localhost", 3306, "cómics" , root, root

E por último dispoñemos do botón de Cancelar, se é que desexamos cancelar o proceso de configuración e volver á pantalla de Inicio.
